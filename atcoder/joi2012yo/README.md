# E

- 模範解答
  - 建物のない領域を地図の外側から塗りつぶす
  - この際に、外側から幅優先探索などを用いて、建物内に存在する領域が塗られないように注意
  - 塗りつぶし後、境界の長さを求めればよい
- suntory
  - 幅優先探索を利用しながら、各建物の建物以外に面している境界を合計していく
  - 上記手法では、建物の中に存在する領域も外周としてカウントされている
  - 一通り上記手法で数えたのち、再度建物のない領域に対して、幅優先探索を行い建物と接している境界を合計する
  - その際に、地図の壁に到達した場合、その合計は利用しない
  - 壁への到達がなかったものは、建物の中に存在する領域になるため、建物を探索していた時の合計から引き算する
