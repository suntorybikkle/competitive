# D

- 星座の座標を一つ決定すれば、星座を構成するほかの星の座標は決定する
- 各星の位置に対して、一つを固定しほかの星が存在することができるかすべて判定すればよい
- 全探索で計算量は十分に間に合う n (全星の数) * m (星座の星の数)

# E
